1\) Clasificación de edad  

2\) Día de la semana  

3\) Verificación de acceso (inicio de sesión)



¿Para que se utilizan los operadores lógicos en programación?

Los operadores lógicos nos sirven para combinar, comparar, o negar condiciones dentro de nuestro programa.

Nos permite evaluar si varias condiciones se cumplen al mismo tiempo, gracias a esos operadores podemos tomar decisiones dentro de nuestro programa.



¿Por que es importante declarar correctamente el tipo de dato de una variable?

Es importante declarar correctamente el tipo de dato por que porque este nos ayuda a declarar que tipo de valores pueden almacenar una variable y que operaciones no solo matemáticas se pueden realizar con ella.
